#serve-*here* [![Build Status](https://travis-ci.org/vivaxy/here.svg?branch=master)](https://travis-ci.org/vivaxy/here)

local static server

## installation

`sudo npm install -g serve-here`

## usage

In your local folder, type `here` and it goes\!

## advanced usage

`here -p 8888`

set port to 8888
